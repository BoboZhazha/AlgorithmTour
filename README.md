# AlgorithmTour



| 算法     | 时间复杂度 | 备注                                    |
| -------- | ---------- | --------------------------------------- |
| 冒泡排序 | O(n^2)     | 每一轮找到一个当前最小的值              |
| 选择排序 | O(n^2)     |                                         |
| 插入排序 | O(n^3/2)   | 对于已经近乎有序的,可以降到O(N)的复杂度 |
| 归并排序 | O(n*logn)  | 自底向上                                |
| 快速排序 | O(n*logn)  |                                         |
